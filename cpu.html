Now let's look at statistical models examining the relationship between price and the features. We will use a multiple linear regression. Before we run the regression, let's see if we have multicollinearity between any of the features. Multicollinearity determines if any features have overlap with each other, meaning one variable can linearly predict other variable(s).¶

In [10]:
def calc_vif(X):

    # Calculating VIF
    vif = pd.DataFrame()
    vif["variables"] = X.columns
    vif["VIF"] = [variance_inflation_factor(X.values, i) for i in range(X.shape[1])]

    return(vif)

X = df.iloc[:,([1,2,3,4,6])] #Not include cd because it is a binary factor that we will look at later
calc_vif(X)

Given the VIF's it appears the screen feature has high multicollinearity with the other features. This means the screen feature has a lot of overlap with the other features and we do not want to include 'screen' into the model because it will be explaining more than it should becuase we are double dipping in prediciting price as indicated by the high VIF. Typically, a VIF near 1 is good, VIF between 1-5 is moderate, and VIF 5+ indicates a lot of overlap between features.¶

4. Conclusion¶

Overall the data project indicates all the features drive computer price, however some features are more important than others. At first glance, one may inclined to think having a cd player is an important factor, however cd's are outdated and having a cd player in a computer is become more and more rare. When computer companies it allowed them to build in more of other features in computers. Based on the regressions, it can be seen that computer companies increased the RAM when they removed the cd player. The additional RAM not only helps the processing capability of the computer but also is a good indicator of the price of a computer.
